FLA HathiTrust Collections

This webpage is a simple demonstration of sharing HathiTrust collections using CSV on the Web (CSVW). It is meant to illustrate how publishing a small amount of metadata along with URL addressable CSV files can enable downstream linked data curation workflows. This work is part of the Workset Creation for Scholarly Analysis project, which is a collaboration between MITH and the HathiTrust Research Center

Both HathiTrust and the HathiTrust Research Center allow users to build collections (or worksets) which are user defined aggregations of HathiTrust resources meant to enable research. It is currently possible to download these collections from HTRC as CSV. What if these CSVs were published with enough metadata to document the semantics of the columns, and with enough provenance information for other systems to track where the data has come from?

The three HathiTrust collections below were created by members of Foreign Literatures in America team at the University of Maryland. The collection metadata was exported to CSV with an accompanying CSVW JSON file that describes the collection using hathitables.
